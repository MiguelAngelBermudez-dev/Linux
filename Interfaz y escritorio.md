## El escritorio grafico

hay dos formas de usarlo:

- Por CLI (interfaz de lineas de comandos)
    - Esto se usa para tareas que requieren hacerse mas rapidamente y que sean repetitivas
- Por GUI (grafical user interface)
    - Es mucho mas facil e intuitivo

## Sistema x Windows

Se carga como uno de los poasos finales en el proceso de arranque

- Display Manager(Gestor de pantallas):
    
    - realiza un segimiento de las pantallas y carga el servidor x, gestiona los inicios de sesion grafico e inicia el entordo de escritorio adecuado
        
    - Imagen de inicio grafico
	    - [[7.png]]
        
        
    
    Para iniciar el gestor de pantallas si no lo tenemos o hay un problema usaremos el comando **startx** O bien, puedes iniciar el gestor de pantalla concreto (**gdm**, **lightdm**, **kdm**, **xdm**, etc.)
    
    ## Inicio de GUI
    
    El gestor de visualizacion para GNOME ES **gdm** para otros es **lightdm** y para kde es **kdm**
    
    ## **gnome-tweaks**
    
    **gnome tweaks** (ajustes de gnome), que da acceso a muchas más opciones de configuración. También te permite instalar fácilmente extensiones de terceros.
    
    ## **gnome-extensions-app**
    
    Es la nueva version para personalizar y configurar temas
